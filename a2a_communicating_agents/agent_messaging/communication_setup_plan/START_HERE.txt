QUICK START (RECOMMENDED)
================================================================================

1. Copy this entire directory to where you want the system:

   cp -r communication_setup_plan /your/destination/

2. Go there:

   cd /your/destination/communication_setup_plan

3. Run the setup script (it installs to current directory):

   ./migrate_agent_system.sh

4. Follow the prompts. Done!

================================================================================
HOW THE SCRIPT WORKS
================================================================================

The script runs from the CURRENT DIRECTORY by default:

  ./migrate_agent_system.sh              # Installs HERE (current directory)

  ./migrate_agent_system.sh /other/path  # Installs THERE (specified path)

It will:
  ✓ Copy all 27 system files
  ✓ Create Python virtual environment
  ✓ Install all dependencies
  ✓ Create configuration files
  ✓ Verify everything works

Time: 15-20 minutes
