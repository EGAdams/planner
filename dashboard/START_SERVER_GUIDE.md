# How to Start the Admin Dashboard Server

This guide provides instructions on how to start the admin dashboard server.

## 1. Build the Project

Before starting the server, you need to build the project. This will compile the TypeScript files and create the necessary JavaScript files.

To build the project, run the following command:

```bash
npm run build
```

## 2. Start the Server

Once the project is built, you can start the server.

To start the server, run the following command:

```bash
npm start
```

The server will start in the foreground, and you will see output in your terminal.

## 3. Access the Dashboard

The admin dashboard is accessible at the following URL:

[http://127.0.0.1:3030](http://127.0.0.1:3030)

You can open this URL in your web browser to view the dashboard.

## 4. Running in the background

If you want to run the server in the background, you can use the following command:

```bash
npm start &
```
